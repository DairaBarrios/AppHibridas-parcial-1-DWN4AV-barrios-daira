Primer Parcial App Hibridas

RBooks

"La API tiene la utilidad de gestionar libros para una biblioteca. Simplifica la carga, edición, eliminación y filtrado de libros. El usuario puede agregar, actualizar y eliminar libros fácilmente. Además, puede filtrar libros según el género o fecha de publicación, simplificando así la búsqueda y selección de lecturas específicas."

Endpoints

Login

POST /login

Permite iniciar sesión en la aplicación.

Body:

  • email: string

  • password: string

Autores

Se requiere mandar por header auth el jwt.

  • GET /autores

    Devuelve la lista de autores paginados.

    Query params:
    • nombre
    • nacionalidad
    • fechaInicio
    • fechaFin
    • page
    • limit
    • campoOrdenamiento
    • direccionOrdenamiento

    • aInicio y fechaFin es el rango de busqueda para fecha de nacimiento

  • POST /autores

    Permite agregar un autor.

    Body:

    • nombre: string

    • apellido: string

    • nacionalidad: string

    • fechaNacimiento: string

  • PUT /autores/:id

    Permite actualizar un autor.

    Path params:

    id: id del autor a modificar

    Body:

    • nombre: string

    • apellido: string

    • nacionalidad: string

    • fechaNacimiento: string

  • GET /autores/:id

    Devuelve un autor por id.

    Path params:

    id: id del autor a buscar

Libros

Se requiere mandar por header auth el jwtoken.

  • GET /libros

    Devuelve la lista de libros activos paginados.

    Query params:
    • titulo
    • genero
    • nroPaginasInicio
    • nroPaginasFin
    • page
    • limit
    • campoOrdenamiento
    • direccionOrdenamiento

  • POST /libros

    Permite agregar un libros.

    Body:

    • titulo: string

    • autor: string

    • genero: string

    • isbn: number

    • editorial: string

    • imagen: string

    • fechaPublicacion: date

    • paginas: number

  • PUT /libros/:id

    Permite actualizar un libro.

    Path params:

    id: id del libro a modificar

    Body:

    • titulo: string

    • autor: string

    • genero: string

    • isbn: number

    • editorial: string

    • imagen: string

    • fechaPublicacion: date

    • paginas: number

  • GET /libros/:id

    Devuelve un libro por id.

    Path params:

    id: id del libro a buscar

  • DELETE /libros/:id

    Desactiva un libro por id.

    Path params:

    id: id del libro a desactivar

  • PUT /libros/activar/:id

    Activa un libro por id.

    Path params:

    id: id del libro a activar
